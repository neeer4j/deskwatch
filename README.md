# DeskWatch

DeskWatch is a simple Windows desktop application that tracks the amount of time you spend in each application. It is built with WPF and .NET 10.

## Features
- Tracks active window usage per application
- Displays usage in a simple table
- Start, Stop, and Reset tracking controls
- Data is kept in memory (no persistence yet)

## Usage
1. Run the application.
2. Click **Start** to begin tracking.
3. Click **Stop** to pause tracking.
4. Click **Reset** to clear all tracked data.

## Development
- Built with WPF (.NET 10)
- No external dependencies
- All tracking is local and private

## Planned Features
- Save/load usage data to disk
- System tray support
- Idle detection
- Exclude list for apps/windows
